# Stick Hero - 棍子英雄

一个简单有趣的 H5 小游戏，考验你的距离判断能力！

## 游戏玩法

1. 按住屏幕（或鼠标左键、空格键）伸长棍子
2. 松开屏幕让棍子倒下成为桥梁
3. 如果棍子长度刚好跨到下一个平台，角色就会走过去
4. 如果棍子太短或太长，角色会掉落失败

## 特性

- 🎮 简单易上手的操作
- 📱 支持移动端和桌面端
- 🎯 随机生成的平台，每次游玩都不同
- 🏆 分数和最高分记录
- 🎨 精美的视觉效果
