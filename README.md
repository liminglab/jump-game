# Stick Hero 小游戏 🎮

一个有趣的 H5 小游戏，考验你的距离判断能力！

## 游戏玩法

1. **按住屏幕** - 棍子会不断伸长
2. **松开屏幕** - 棍子会倒下成为一座桥
3. **成功条件** - 桥必须正好跨到下一个平台
4. **失败条件** - 桥太短或太长都会掉落

## 游戏特性

- 🎯 随机生成的平台宽度和间距
- 📊 实时得分系统
- 🏆 最高分记录（本地存储）
- 📱 支持触摸和鼠标操作
- 🎨 精美的渐变背景和动画效果
- 🔄 无限关卡，难度逐渐增加

## 如何开始

点击"开始游戏"按钮，然后按住屏幕伸长棍子，松开后观察棍子是否成功连接到下一个平台。

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript
- Canvas API 绘制游戏画面
- 响应式设计，适配移动端和桌面端
